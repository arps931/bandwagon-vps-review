## 搬瓦工到底是什么

先给没接触过的人解释一下。

BandwagonHost，也就是大家叫的"搬瓦工"，是加拿大 IT7 Networks 旗下的 VPS 品牌，做了二十年了。之所以叫搬瓦工，就是英文发音"ban-wag-on"听着像，后来就叫开了。

价格不便宜，但它解决的问题很明确：**国内访问海外服务器速度慢、晚高峰丢包严重**。这正是很多人买 VPS 最头疼的地方。

## 三大套餐系列，哪个适合你

搬瓦工目前把套餐分为三大系列：**Basic VPS**、**E-Commerce VPS**、**Ultra VPS**。名字改过几次，但对应的定位没变。

### Basic VPS（入门系列）

就是以前大家说的 KVM 套餐。入门年付 $49.99，1Gbps 带宽，SSD RAID-10 存储，可选机房 7 个，买完后可以在 KiwiVM 面板里切换。

这个系列的线路质量一般，走的是普通三网直连，不是 CN2 GIA。用来学 Linux、跑个人博客、测试项目完全够，但如果你要求晚高峰速度稳定，Basic 系列大概率让你失望。

### E-Commerce VPS（主力系列，最多人买）

对应以前的 CN2 GIA-E 套餐。这是搬瓦工卖得最好的一条线，用的人也最多。

入门配置年付 $169.99（季付 $49.99），带宽 2.5Gbps 起步，支持 12 个以上机房自由切换，包括洛杉矶 DC6、DC9、日本软银、荷兰联通 9929 等。回程三网优化：电信走 CN2 GIA，联通走 AS9929，移动走 CMIN2。

实际体感就是，换不同机房真的像换了一台机器。如果 DC6 某段时间有抖动，切到 DC9 或者软银，体感立刻不一样。

### Ultra VPS（高端系列）

主要是香港 CN2 GIA、日本大阪 CN2 GIA、日本东京 CN2 GIA 这三个机房。

物理距离近，延迟可以低到 9ms（香港）到 30ms 左右（东京/大阪），三网回程全程 CN2 GIA。代价是价格高——香港套餐月付 $89.99 起，日本大阪月付要看具体配置，比 E-Commerce 贵一倍多。

对延迟不敏感的场景，E-Commerce 的 150-160ms 实际上用不出 Ultra 的区别。Ultra 真正有用的，是做实时性极强的业务，或者就是不差钱。

## 搬瓦工好用吗，用户真实体验的几个维度

### 稳定性

这块没什么争议，是搬瓦工口碑里被提到最多的优点。

我自己这段时间基本没遇到过宕机，顶多是某次机房维护窗口提前收到邮件，自己手动切了一下机房，数据完全保留，迁移花了不到一分钟。SLA 承诺是 99.9%，实际跑下来感觉比这个数字还稳。

免费快照和自动备份是标配。不少同价位的商家这两个功能要单独收费，搬瓦工是全系列自带。做过一次重大配置变更，快照及时救了场，这种省心感是真实的。

### KiwiVM 控制面板

这个面板是搬瓦工自研的，也是很多人说它好用的原因之一。

开机关机重装系统是最基本的。更好用的是机房迁移——后台直接选目标机房，几分钟切完，IP 同步换，不需要工单，不需要等客服。对于想测试不同机房速度的人来说，这比其他很多家方便太多了。

流量、CPU 占用、网络 I/O 都有实时监控图表，快照备份都在这里管理。界面是英文的，但逻辑简单，大多数操作照着点就能搞定。

### 速度实测体感

CN2 GIA-E 洛杉矶 DC6 机房，三网测下来都能跑到应有的速度。电信晚高峰下载没有明显限速，联通那边表现同样稳。移动接入 CMIN2 线路，延迟控制也不错。

普通操作比如 SSH 敲命令、Git 拉代码，完全没有卡顿感。跑 WordPress + 宝塔面板，1GB 内存版本会偏紧，但页面加载还是利索的，2GB 会宽裕很多。

### 缺点也要说清楚

说句实话，搬瓦工不是完美的。

IP 被墙之后换 IP 要额外付 $8，这个不便宜，被墙频率高的话会是一笔不小的开销。遇到这种情况，切机房会自动换 IP，算是曲线解法，但要消耗和硬盘大小相当的流量。

CPU 性能一直被人吐槽，同价位的性能测试跑出来属于中等偏下水平。不过从 Box 系列限量版开始，搬瓦工的新机型逐渐换成了 AMD EPYC 系列，DC9 机房也升级了 NVMe SSD，这块有改善趋势，但常规套餐还是偏老架构。

价格也没什么谦虚的空间——年付 $169.99 的 CN2 GIA-E 入门套餐，对于预算有限的人确实有点重，算下来一个月差不多 14 刀。讲真，纯论价格，同价位你可以找到配置更高的 VPS，但线路质量和稳定性不一定能保证。

## 常见问题

**Q：搬瓦工支持退款吗？**  
支持。30 天内不满意可以申请全额退款，走官方工单。有朋友试用了将近一个月觉得不合适，退款审批下来了，没有额外扣费。不过这个政策有次数限制，不是无限次的。

**Q：搬瓦工好用吗，适合科学上网吗？**  
搬瓦工卖的是 VPS，不是成品代理服务，需要自己在服务器上搭建代理软件。从 VPS 本身的角度来说，CN2 GIA-E 的线路质量能满足大多数科学上网场景，稳定性也有保障。

**Q：CN2 GIA 和普通线路区别多大？**  
明显很多。普通线路晚高峰延迟容易飙到 200ms 以上，丢包也会上升；CN2 GIA 晚高峰延迟基本维持在 160ms 左右，丢包极低。如果你经常在晚上用，这个差距很实在。

**Q：买了 Basic 套餐以后能升级到 E-Commerce 吗？**  
可以升级，但方向是单向的——只能升不能降。升级时系统会按剩余使用时长折算价差，不会浪费已付的费用。如果第一次买不确定需求，可以先从 Basic 起，用着合适再加钱升级。

**Q：流量 1TB 够用吗？**  
对于个人建站或轻度代理使用，1TB/月一般够。如果做大流量传输或者当主力下载机，可能几天就用掉了，建议选流量更大的套餐或者动态评估用量。
